📊 Task 03: HR Employee Attrition Dashboard.

📌 Project Overview

This project involved analyzing an HR Employee Attrition dataset using Microsoft Power BI Desktop. 
The goal was to transform raw employee data into an interactive dashboard that provides insights into employee turnover and key factors influencing attrition. 
Developed as part of my Data Analyst Internship at SkillCraft Technology.

🎯 Key Objectives:-

Data Preparation:
Processed and structured the dataset to ensure proper formatting and usability for analysis.

Data Visualization:
Created multiple visualizations such as Pie Charts, Bar Charts, and Column Charts to analyze attrition patterns.

Interactivity:
Implemented slicers for Department, Age Group, and Gender to enable dynamic filtering.

Insight Generation:
Analyzed trends related to salary, age, department, and job roles to understand employee attrition behavior.

📊 Dashboard Features

Attrition Overview (Pie Chart)

Attrition by Department

Attrition by Age Group

Attrition by Salary Group

Attrition by Job Role

Interactive Filters (Slicers)

💡 Key Insights

Based on the analysis of the HR dataset:

Age-Based Attrition:
Employees in their 30s show the highest attrition, indicating mid-career transitions.

Salary Impact:
Lower salary groups have significantly higher attrition rates compared to higher income groups.

Department Trends:
Certain departments (e.g., Sales/HR) experience higher employee turnover.

Job Role Analysis:
Specific job roles contribute more to attrition, highlighting role-based dissatisfaction or workload issues.

🚀 Outcome

This project demonstrates the ability to use Power BI for building interactive dashboards, analyzing HR data, and deriving meaningful business insights. 
It highlights how data visualization can support decision-making in employee retention strategies.
